Python is my most recent undertaking so there is not as much here compared to others, but for now there will be a "remake" of space invaders.

  • Alien Invasion Which is one of the first games I ever played. It needs the following files to function.
  • Aliens This is to create an alien.
  • Bullet This is used to create the bullets that the ship fires.
  • Button To create the play button on the screen.
  • Game Functions The master file that processes the entire game.
  • Game Stats The file that holds the score.
  • Scoreboard The file that displays your score on the screen.
  • Settings The file that holds all of the speed, ship, alien, and bullet settings
  • Ship The file that holds information about the ship.
